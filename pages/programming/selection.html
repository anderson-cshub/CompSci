Basic if Statement

Runs a block of code only if a condition is True. 

age = 18

if age >= 18:
    print("You can vote.")

Only prints the message when the condition is True.

if and else

Use else when you want something to happen if the condition is False. 

score = 45

if score >= 50:
    print("Pass")
else:
    print("Fail")

if, elif, else

elif lets you check multiple conditions in order. 

temp = 25

if temp > 30:
    print("Hot")
elif temp > 20:
    print("Warm")
else:
    print("Cold")

Multiple elif Conditions

Useful when you need several possible outcomes. 

grade = 72

if grade >= 90:
    print("A")
elif grade >= 80:
    print("B")
elif grade >= 70:
    print("C")
else:
    print("D")

Logic Error Example

Conditions must be ordered correctly. This example contains a logic error. 

age = 65

if age > 50:
    print("Over 50")
elif age > 60:
    print("Over 60")

The second condition can never run — age > 60 is also age > 50. This is a classic GCSE logic error.

Comments

Comments help explain your code. Python uses # for single‑line comments. 

# Ask the user for their age
age = int(input("Enter age: "))

# Check if they are an adult
if age >= 18:
    print("Adult")

Selection (Branching) — Making Decisions in Python

Selection is one of the three key programming constructs:

  • Sequence — instructions run in order
  • Selection — decisions using if
  • Iteration — repeating instructions

Selection allows your program to choose different paths depending on conditions. Python uses if, elif, and else.

Comparison Operators

  • == equal to
  • != not equal to
  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to

Logic Errors

A logic error is when the program runs but produces the wrong output. These often happen when conditions are in the wrong order. 

age = 65

if age > 50:
    print("Over 50")
elif age > 60:
    print("Over 60")

The second condition can never run because age > 60 is also age > 50. OCR love this type of question.
