Python Intro

Before writing full programs, students need to understand the absolute fundamentals:

  • How to print text (“Hello World”)
  • Indentation rules
  • Colons after control statements
  • Basic syntax errors and how to spot them

Hello World

The simplest Python program:

print("Hello World")

This teaches:

  • Function calls
  • Quotation marks
  • Parentheses

Indentation

Python uses indentation instead of braces:

if True:
    print("Indented!")

If indentation is wrong, Python throws an error.
