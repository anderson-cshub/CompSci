Iteration

For Loop (definite iteration)

A for loop repeats a block of code a set number of times. The loop variable (often i or x) changes each time. 

for i in range(5):
    print(i)

This prints 0, 1, 2, 3, 4.

The range() Function

range() generates a sequence of numbers. You can use 1, 2 or 3 arguments. 

range(5)        # 0 → 4
range(2, 6)     # 2 → 5
range(1, 10, 2) # 1, 3, 5, 7, 9

For‑Each Loop

Loops through each item in a list or string. The loop variable becomes each item in turn. 

names = ["Alice", "Bob", "Charlie"]

for name in names:
    print(name)

While Loop (indefinite iteration)

A while loop repeats until a condition becomes False. Used when you don’t know how many times you need to loop. 

count = 0

while count < 5:
    print(count)
    count += 1

Loop Control: break & continue

These commands change how loops behave. 

for i in range(10):
    if i == 5:
        break      # stops the loop
    print(i)

Nested Loops

A loop inside another loop. Useful for grids, tables, and multi‑level data. 

for row in range(3):
    for col in range(3):
        print(row, col)

Iteration — One of the Three Key Constructs

Iteration is one of the three fundamental programming constructs:

  • Sequence — instructions run in order
  • Selection — decisions using if
  • Iteration — repeating instructions

Python supports two main types of iteration:

  • Definite iteration — using for when you know how many times to loop.
  • Indefinite iteration — using while when you loop until a condition changes.

Common GCSE Mistakes

  • Forgetting to update the loop variable in a while loop.
  • Using range incorrectly (off‑by‑one errors).
  • Confusing for‑each loops with range loops.
  • Accidentally creating infinite loops.
