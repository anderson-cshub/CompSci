Lists and Arrays

Lists in Python

A list stores multiple values in one variable. Lists are dynamic — they can grow or shrink. 

scores = [10, 20, 30, 40]
print(scores[0])  # first item
scores.append(50)

Lists can hold mixed data types and change size at runtime.

Arrays (GCSE Concept)

In GCSE theory, an array is a fixed‑size structure storing data of the same type. 

# Example GCSE pseudocode
DECLARE scores[5] AS INTEGER
scores[0] ← 10
scores[1] ← 20

Arrays are fixed in size and type — unlike Python lists.

2D Array — Tic‑Tac‑Toe Grid

A 2D array stores data in rows and columns. Each element is accessed by two indices. 

grid = [
    ["X", "O", "X"],
    ["O", "X", "O"],
    ["X", " ", "O"]
]

print(grid[0][2])  # top‑right cell

Each row represents a line on the board; each column a position.

2D Array — Pupil Scores

A table of names, subjects, and scores can be stored as a 2D array. 

results = [
    ["Alice", "Maths", 88],
    ["Bob", "Science", 75],
    ["Charlie", "English", 92]
]

print(results[1][0])  # Bob
print(results[2][2])  # Charlie’s score

Row = pupil, column = data field.

Accessing Elements

Use indices to access or update elements in lists or arrays. 

scores = [10, 20, 30]
scores[1] = 25
print(scores)

Lists and Arrays — Storing Collections of Data

Lists and arrays are both used to store multiple values, but they behave differently. Python uses lists, while GCSE pseudocode often refers to arrays.

Python Lists

  • Dynamic — can grow or shrink.
  • Can store mixed data types.
  • Access items using indices starting at 0.

GCSE Arrays

  • Fixed size and data type.
  • Used in pseudocode and exam questions.
  • Often shown as tables or grids.

2D Arrays

A 2D array stores data in rows and columns — like a spreadsheet. Each element is accessed using two indices: [row][column]. 

# Tic‑Tac‑Toe grid example
grid = [
    ["X", "O", "X"],
    ["O", "X", "O"],
    ["X", " ", "O"]
]
print(grid[0][0])  # top‑left cell
print(grid[0][2])  # top‑right cell
