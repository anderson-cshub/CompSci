Programming

Python

Python is the world's leading programming language for learning, creativity, and innovation. From AI and robotics to websites and data analysis, Python powers many of the technologies we use every day.

Its clear, readable code makes it the perfect language for beginners and experts alike.

It is also favoured by the OCR, so it makes sense to use what they use! It will align closely to the mark schemes and while learning is the goal, a certificate with a 9 or an A* is what you can show people.

IDE - integrated development environment

To write code you will need an IDE. Python does have its own https://www.python.org/downloads/ here, python idle but I recommend a nicer interface to learn how to program.

MU is a simple interface with some nice debugging features.

  • Extremely simple interface.
  • Great little thumbs-up debugger
  • Different lighting modes to fit your style
  • Very little clutter or distraction.
  • Great for introducing Python in the first few lessons.
  • Supportive for beginners getting started
  • Interface and use can take you all the way through to GCSE
codewith.mu

Thonny

Some students, once they have used MU for a while, move on to Thonny. It is:

  • Built specifically for teaching Python
  • Excellent debugger that shows how variables change step-by-step
  • Clear variable and memory views help students understand program execution
  • Supportive for beginners but grows with students as they become more advanced
  • Better support for packages, virtual environments, and complex projects
