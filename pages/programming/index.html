Programming

Python

Python is the world's leading programming language for learning, creativity, and innovation. From AI and robotics to websites and data analysis, Python powers many of the technologies we use every day.

Its clear, readable code makes it the perfect language for beginners and experts alike. It is also favoured by the OCR, so it makes sense to use what they use! It will align closely to the mark schemes and while learning is the goal, a certificate with a 9 or an A* is what you can show people.

IDE - integrated development environment

To write code you will need an IDE. Python does have its own https://www.python.org/downloads/ here, python idle but I recommend a nicer interface to learn how to program.

MU is a simple interface with some nice debugging features. codewith.mu

Thonny

Some students once they have use MU for a while move on to Thonny. Built specifically for teaching Python. Excellent debugger that shows how variables change step-by-step. Clear variable and memory views help students understand program execution. Supports beginners but can grow with students as they become more advanced. Better support for packages, virtual environments, and more complex projects. https://thonny.org/
